# EcoCompare Tool

## Project Overview
EcoCompare is a simple web-based tool designed to help users understand the relationship between cost and environmental sustainability. Users can select two different energy sources or packaging materials to see a side-by-side comparison of their price, eco-score, and the specific reasons behind their environmental impact.

## Core Features
* **Comparison Calculator**: Allows users to select two items from a dropdown menu for instant analysis.
* **Sustainability Metrics**: Displays an Eco-Score out of 10 to rank how "green" an item is.
* **Cost Analysis**: Provides a high-level estimate of the financial cost (e.g., Low, Medium, or High Setup).
* **Educational Reasoning**: Includes a "Reasoning" section that explains why an item received its specific score.
* **Dynamic Verdict**: Automatically calculates and highlights the "winner" based on the sustainability score.

## Comparison Options
The tool currently supports the following items:
* Solar Energy
* Coal Energy
* Wind Power
* Plastic Packaging
* Glass Packaging

## Technologies Used
* **HTML5**: Used for the structure of the website and the data table.
* **CSS3**: Used for the "Nature" theme (light green and sky blue colors) and the responsive layout.
* **JavaScript**: Used to handle the comparison logic and update the results on the screen without refreshing the page.

## How to Run the Project
1.  Download the `index.html` file.
2.  Open the file in any modern web browser (Chrome, Firefox, Edge, or Safari).
3.  Select two items from the dropdown menus and click the "Compare Now" button.
